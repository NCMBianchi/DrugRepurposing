About Page

A MSc project by Niccolò Bianchi, based on the works of Carmen Reep (DrugRepurposing) and Nuria Queralt Rosinach (BioKnowledge reviewer).

This updated drug repurposing app finds candidate compounds for a symptom of a specific rare disease –emphasising the role of negative associations.
For this, it uses a workflow that has five main steps:

  1. Creation of an RDF knowledge graph by selecting a subnetwork from Monarch Initiative's new v3 API (i.e. FastAPI), adding drugs using the DGIdb database, creating drug-drug edges based on drug compound structure similarity.

  2. Graph embedding using RDF2Vec.

  3. Creation of edge representations (=training/test data) by fusing drug and gene feature vectors.

  4. Training a [SURE machine learning model in order to assess information on negative links between edges, to account for information regarding associations that are known to be absent].

  5. Training an XGBoost machine learning model which is evaluated using repeated stratified 10-fold cross validation, where the best model is used to predict unknown drug-gene interactions of interest.
